# 宅建パス 🏡

宅地建物取引士試験の過去問トレーニングアプリ(令和2〜7年度・全8回400問、全問解説付き)。

- 分野別演習(権利関係/法令上の制限/税・その他/宅建業法/免除科目)
- 弱点克服(間違えた問題・ブックマークを自動収集)
- 模擬試験(年度別8回分+ランダム編成、120分タイマー、合格判定)
- 学習履歴はブラウザ内(localStorage)に保存。サーバー送信なし

単一HTMLファイルで動作します。スマートフォンのブラウザでそのまま利用できます。

出典: 一般財団法人 不動産適正取引推進機構が公表する試験問題・正解番号を基に作成。
問題文中の法令は各年度の出題時点の規定に基づきます。解説は独自に作成したものです。
